### May 6, 2026

**Topics studied:**
- Cloud concepts and connectivity
- Cloud deployment models
- Cloud service models
- Virtual networks
- Network Function Virtualization (NFV)
- Virtual Private Cloud (VPC)
- Cloud gateways
- Cloud security groups and network security lists

**Resources used:**
- Professor Messer Network+ videos
- Official CompTIA Network+ N10-009 exam objectives

**What I learned:**
- I learned that cloud computing provides on-demand access to resources such as servers, storage, networking, and applications.
- I learned the difference between elasticity and scalability.
- I learned that multitenancy allows multiple customers to use the same cloud infrastructure while keeping their environments separate.
- I learned that virtual networks allow virtual servers and cloud resources to communicate like they are connected through physical networking equipment.
- I learned that NFV replaces physical network devices with virtual versions, such as virtual routers, switches, load balancers, and firewalls.
- I learned that a VPC is a private pool of resources created inside a public cloud.
- I learned that transit gateways can connect VPCs together and act like cloud routers.
- I learned that VPNs can connect users or on-premises networks to cloud environments securely.
- I learned that internet gateways, NAT gateways, and VPC endpoints provide different ways for cloud resources to communicate.
- I learned that cloud security groups and network security lists act like firewalls for cloud environments.
- I learned that network security lists are broader because they usually apply to an entire subnet.
- I learned that network security groups are more granular because they can apply to specific virtual network interfaces or resources.
- I learned the difference between public, private, and hybrid cloud deployment models.
- I learned the difference between SaaS, IaaS, and PaaS.
- I learned that SaaS provides complete software applications, IaaS provides cloud infrastructure, and PaaS provides a development platform.

**Things to review:**
- Difference between elasticity and scalability.
- Difference between public, private, and hybrid cloud models.
- Difference between SaaS, IaaS, and PaaS.
- Difference between security groups and network security lists.
- Difference between internet gateways, NAT gateways, and VPC endpoints.
- How transit gateways connect VPCs.
- How NFV replaces physical networking devices.
- Difference between public, private, and hybrid cloud.
- Difference between SaaS, IaaS, and PaaS.
- What the customer manages in each cloud service model.
- What the cloud provider manages in each cloud service model.
- Why security responsibilities change depending on the cloud model.
