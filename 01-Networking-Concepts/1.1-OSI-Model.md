# Networking Concepts

## 1.1 Understanding the OSI Model

### Summary
The OSI model is a 7-layer framework used to explain how data moves across a network.

### Key Terms
- **Physical layer:** Handles signaling, cables, connectors, and physical transmission.
- **Data Link layer:** Handles local network communication, MAC addresses, and frames.
- **Network layer:** Handles IP addressing and routing.
- **Transport layer:** Handles end-to-end communication using TCP or UDP.
- **Session layer:** Manages communication sessions.
- **Presentation layer:** Handles formatting, encryption, and data translation.
- **Application layer:** Provides network services used by applications, such as HTTP, HTTPS, DNS, FTP, SMTP, and DHCP.

### Important Details
- TCP/IP is the practical networking model used in real networks.
- OSI is mainly used as a learning and troubleshooting framework.
- Encapsulation means data gets wrapped with headers as it moves down the layers.

### Example
When visiting a website:
1. The application uses HTTPS.
2. TCP handles reliable transport.
3. IP handles addressing and routing.
4. Ethernet or Wi-Fi handles local delivery.
5. Cables or radio signals carry the bits.
